# Checklist de prepublicación

Antes de recomendar publicar, verifica:

- [ ] **Título** claro y específico (no genérico).
- [ ] **Resumen** de 1 párrafo que refleje el contenido.
- [ ] **Estructura** con secciones y transiciones lógicas.
- [ ] **Claridad**: frases directas, sin jerga innecesaria.
- [ ] **Coherencia**: el hilo argumental se sostiene de principio a fin.
- [ ] **Sin redundancias** ni repeticiones evitables.
- [ ] **Ortografía y gramática** correctas.
- [ ] **Originalidad**: aporta algo frente a lo ya publicado en ReadHub.
- [ ] **Sin contradicciones** no justificadas con otros artículos.
- [ ] **Palabras clave** identificadas.
- [ ] **Fuentes** de ReadHub citadas cuando corresponda.

Para cada punto no cumplido, da una recomendación accionable.
